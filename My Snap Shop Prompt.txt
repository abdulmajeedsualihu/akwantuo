## 1. PRODUCT ARCHITECTURE

### **Core Concept**
**"WhatsApp-style simplicity meets AI-powered professionalism"**

Vendors snap photos → AI auto-generates professional storefront → One-tap publishing → Shareable link (works on
2G) + WhatsApp integration

### **Key Features**

**For Vendors (Mobile-First):**
- **Photo-to-Store**: Snap product pics → AI extracts details (price estimation, category, description in
Twi/English)
- **Smart Templates**: AI suggests 3 design styles based on product type (Artisan vs. Food vs. Services)
- **Auto-Content**: Generates SEO-friendly descriptions, sizing guides, "About the Maker" stories
- **One-Click Publish**: Generates lightweight PWA (Progressive Web App) storefront
- **WhatsApp Integration**: Auto-generates catalog messages for WhatsApp Business

**AI Recommendation Engine:**
- **Color Psychology**: Suggests palettes based on product (e.g., earth tones for crafts, vibrant for food)
- **Layout Alternatives**: Grid vs. List vs. Story format
- **Pricing Suggestions**: "Similar items sell for 45-60 GHS"
- **Info Completeness**: "Add care instructions to increase trust by 40%"

### **Tech Stack (Ghana-Optimized)**

```
Frontend: React + Tailwind (lightweight, CDN-hosted)
Backend: Node.js/Express or Firebase (real-time updates)
AI Layer:
  - Gemini Pro Vision (image understanding, free tier generous)
  - Claude Haiku (text generation, fast)
Storage: Cloudinary (image optimization for slow networks)
Payments: Paystack/Flutterwave integration
Hosting: Vercel (edge network, fast in Accra/Lagos)
```

**Critical Constraints:**
- **< 500KB initial load** (2G compatibility)
- **Offline-first** (PWA with service workers)
- **USSD fallback** for feature phone users (via Twilio Africa)

```javascript
// Pseudo-code for the AI generation flow
async function generateStorefront(photos, businessType) {
  // Step 1: Analyze images
  const imageAnalysis = await geminiVision.analyze(photos, {
    prompt: "Identify product type, suggested price in GHS, materials, and generate 3 description options (formal,
friendly, Twi-English mix)"
  });

  // Step 2: Select template based on category
  const templates = {
    'food': ['vibrant', 'clean', 'rustic'],
    'crafts': ['cultural', 'minimal', 'artisan'],
    'services': ['professional', 'modern', 'trust']
  };

  // Step 3: Generate color palettes
  const colors = await generatePalette(imageAnalysis.dominantColors);

  // Step 4: Build React component code
  return compileToReact({
    template: templates[businessType][0],
    colors: colors,
    content: imageAnalysis,
    optimizations: '2g-friendly'
  });
}
```

## 6. BACKEND ARCHITECTURE (Ghana-Optimized)

### **System Design**
```
Vendor Phone (2G/3G)
    ↓
CDN (Cloudflare/Vercel Edge - Lagos node)
    ↓
API Gateway (Rate limiting, MoMo validation)
    ↓
Microservices:
├── AI Service (Gemini Pro Vision + Claude)
├── Store Generator (Static site compiler)
├── Payment Webhook (Paystack/Flutterwave)
└── SMS/WhatsApp Gateway (Twilio Africa)
    ↓
Storage:
├── PostgreSQL (Vendor profiles, orders)
├── Cloudinary (Images with auto-optimization)
└── S3/Backblaze (Generated static sites)
```

### **Database Schema (Simplified)**

```sql
-- Vendors table (optimized for informal economy)
CREATE TABLE vendors (
    id UUID PRIMARY KEY,
    phone_number VARCHAR(15) UNIQUE, -- +233 format, primary ID
    business_name VARCHAR(100),
    category ENUM('food', 'crafts', 'clothing', 'services', 'electronics'),
    location POINT, -- PostGIS for "near me" features
    template_id VARCHAR(50),
    color_palette JSON, -- {primary: "#059669", accent: "#F59E0B"}
    is_premium BOOLEAN DEFAULT false,
    whatsapp_business_id VARCHAR(50),
    created_at TIMESTAMP,
    -- Critical for Ghana: offline sync support
    last_sync TIMESTAMP,
    device_fingerprint VARCHAR(255) -- fraud prevention
);

-- Products with AI-generated metadata
CREATE TABLE products (
    id UUID PRIMARY KEY,
    vendor_id UUID REFERENCES vendors(id),
    images JSON, -- Array of Cloudinary URLs with transformations
    ai_description TEXT,
    ai_tags JSON, -- ["kente", "handmade", "accra"] for search
    price_ghs DECIMAL(10,2),
    is_negotiable BOOLEAN DEFAULT true, -- Ghana cultural necessity
    stock_status ENUM('available', 'low', 'made_to_order'),
    generated_html TEXT, -- Cached storefront HTML
    views_count INT DEFAULT 0
);

-- Orders (MoMo integration critical)
CREATE TABLE orders (
    id UUID PRIMARY KEY,
    product_id UUID,
    customer_phone VARCHAR(15),
    amount_ghs DECIMAL,
    status ENUM('pending_momo', 'paid', 'shipped', 'delivered'),
    momo_transaction_id VARCHAR(100),
    momo_network ENUM('mtn', 'vodafone', 'airtel_tigo'),
    created_at TIMESTAMP
);
```

### **AI Service Implementation**

**Image Analysis Pipeline** (Node.js/Express):

```javascript
// services/aiGenerator.js
const { GoogleGenerativeAI } = require("@google/generative-ai");
const genAI = new GoogleGenerativeAI(process.env.GEMINI_KEY);

async function analyzeProductImages(imageBuffers) {
  const model = genAI.getGenerativeModel({ model: "gemini-pro-vision" });

  const prompt = `
    Analyze these product photos from a Ghanaian market vendor.
    Return JSON with:
    - product_name: specific name (e.g., "Kente Cloth" not just "Fabric")
    - category: one of [food, textiles, crafts, woodwork, services]
    - suggested_price_ghs: estimated fair market price in Ghana Cedis
    - description_options: {
        formal: "Professional description for tourists",
        friendly: "Warm description emphasizing handmade quality",
        local: "Description mixing English and Twi phrases"
      }
    - color_palette: {
        primary: hex color from product,
        secondary: complementary color,
        accent: gold/earth tone
      }
    - key_features: array of 3 selling points
    - care_instructions: if applicable

    Consider Ghanaian context: emphasize authenticity, handmade quality, and cultural significance.
  `;

  const result = await model.generateContent([prompt, ...imageBuffers]);
  const response = await result.response;
  return JSON.parse(response.text());
}
```

## 7. MOBILE MONEY INTEGRATION (Critical for Ghana)

### **The "Paystack + WhatsApp" Flow**

Since most Ghanaian vendors already use MoMo but expats/tourists want card payments:

```javascript
// Payment orchestration
async function processPayment(order, customerType) {
  if (customerType === 'local') {
    // Direct MoMo push
    return await paystack.chargeMobileMoney({
      phone: order.customer_phone,
      amount: order.amount_ghs * 100, // pesewas
      network: detectNetwork(order.customer_phone), // 024/054 = MTN, etc.
      metadata: { vendor_id: order.vendor_id }
    });
  } else {
    // Card/Apple Pay for expats, settle to vendor's MoMo
    return await paystack.chargeCard({
      amount: order.amount_ghs * 100,
      email: customer.email,
      // Vendor receives MoMo notification within 30 seconds
    });
  }
}
```

**Vendor Notification Flow**:
1. Order comes in → Instant WhatsApp Business message to vendor
2. "New order: 2x Kente scarves (GHS 120). Reply CONFIRM to accept."
3. Vendor replies CONFIRM → Customer charged → Money hits vendor's MoMo in T+1

## 12. RISK MITIGATION

**Regulatory Risk**:
- **Bank of Ghana**: If holding funds, need payment service provider license
- **Mitigation**: Use Paystack/Flutterwave as the merchant of record, you just facilitate

**Fraud Risk**:
- Fake vendors taking money, not delivering
- **Mitigation**: "Verified Vendor" badge requires video call + ID check + physical address verification (send
someone to the market)

**Competition**:
- Jumia, Tonaton exist but don't serve informal vendors well
- **Differentiation**: AI generation + WhatsApp-native workflow (not app-based)

### **Technical Architecture**

```javascript
// services/voiceProductService.js
const { SpeechClient } = require('@google-cloud/speech');
const { Translate } = require('@google-cloud/translate');
const ffmpeg = require('fluent-ffmpeg');

class VoiceToProduct {
  constructor() {
    this.speechClient = new SpeechClient();
    this.translator = new Translate();

    // Ghana-specific language codes
    this.languageMap = {
      'twi': 'tw',
      'ga': 'gaa',
      'ewe': 'ee',
      'english': 'en-GH',
      'pidgin': 'en-GH' // Fallback to English with specific prompts
    };
  }

  async processVoiceNote(audioBuffer, phoneNumber) {
    // Step 1: Detect language (Ghana has 80+ languages)
    const detectedLang = await this.detectLanguage(audioBuffer);

    // Step 2: Transcribe with context
    const transcription = await this.transcribeWithContext(
      audioBuffer,
      detectedLang,
      'product_description' // Context hint
    );

    // Step 3: Extract structured data using AI
    const productData = await this.extractProductEntities(transcription, detectedLang);

    // Step 4: Generate confirmation audio (text-to-speech)
    const confirmationAudio = await this.generateConfirmationTTS(
      productData,
      detectedLang
    );

    // Step 5: Send back via WhatsApp or SMS link
    await this.sendConfirmation(phoneNumber, productData, confirmationAudio);

    return productData;
  }

  async transcribeWithContext(audioBuffer, language, context) {
    // Convert WhatsApp audio (OGG/Opus) to FLAC
    const flacBuffer = await this.convertToFlac(audioBuffer);

    const config = {
      encoding: 'FLAC',
      sampleRateHertz: 16000,
      languageCode: this.languageMap[language] || 'en-GH',
      alternativeLanguageCodes: ['en-GH', 'tw'], // Code switching common
      model: 'latest_long',
      useEnhanced: true,
      speechContexts: [{
        phrases: [
          'cedis', 'Ghana', 'Accra', 'Kumasi', 'Kente', 'fufu', 'banku',
          'made in Ghana', 'authentic', 'handmade', 'price', 'negotiable',
          'small', 'medium', 'large', 'color', 'red', 'black', 'gold'
        ],
        boost: 20.0 // Increase recognition of local terms
      }]
    };

    const [response] = await this.speechClient.recognize({
      audio: { content: flacBuffer.toString('base64') },
      config: config
    });

    return response.results
      .map(result => result.alternatives[0].transcript)
      .join('\n');
  }

  async extractProductEntities(transcription, language) {
    // Use Claude to extract from local language transcription
    const prompt = `
      Extract product information from this ${language} speech transcription.
      Transcription: "${transcription}"

      Return JSON:
      {
        "product_name": "specific name",
        "price_mentioned": number or null,
        "currency": "GHS",
        "category": "food|textiles|crafts|services",
        "attributes": {
          "color": [],
          "size": [],
          "material": []
        },
        "negotiation_hint": boolean, // Did they say "we can talk" or "make offer"?
        "location_mentioned": string,
        "urgency": "high|medium|low"
      }

      Note: In Ghanaian markets, prices are often not stated upfront ("come and see").
      If price is missing, set price_mentioned to null and negotiation_hint to true.
    `;

    const response = await anthropic.messages.create({
      model: "claude-3-5-sonnet-20241022",
      messages: [{ role: "user", content: prompt }]
    });

    return JSON.parse(response.content[0].text);
  }

  // Text-to-Speech for confirmations (local languages)
  async generateConfirmationTTS(productData, language) {
    // Use Google Cloud TTS with Chirp voices (supports African languages)
    const request = {
      input: { text: this.buildConfirmationText(productData, language) },
      voice: {
        languageCode: this.languageMap[language],
        name: `${this.languageMap[language]}-Standard-A`,
        ssmlGender: 'FEMALE' // Research shows female voices trusted more in Ghana for commerce
      },
      audioConfig: {
        audioEncoding: 'MP3',
        speakingRate: 0.9 // Slightly slower for clarity
      }
    };

    const [response] = await this.speechClient.synthesizeSpeech(request);
    return response.audioContent;
  }

  buildConfirmationText(data, lang) {
    // Localized confirmation messages
    const messages = {
      'tw': `Wo nneɛma no ne ${data.product_name}. Boɔ no yɛ ${data.price_mentioned} sidi. Sɛ ɛyɛ ture a, hyɛ 1.
Sɛ ɛnnyɛ ture nso a, hyɛ 2.`,
      'ga': `Nyɛmi lɛ ${data.product_name}. Shiɔ lɛ ${data.price_mentioned} Ghana cedis. Baŋŋ lɛ, lɛba 1. Daŋŋ lɛ,
lɛba 2.`,
      'en': `Your product is ${data.product_name}. The price is ${data.price_mentioned} Ghana Cedis. If this is
correct, press 1. If wrong, press 2.`
    };
    return messages[lang] || messages['en'];
  }
}
```

## 20. MULTI-LANGUAGE NLP (Twi, Ga, Ewe, Pidgin)

### **The Challenge**
- **Code-switching**: Ghanaians mix English + Local language in same sentence ("The price is ten cedis, ɛyɛ dɛ")
- **No standard spelling**: "Kente" vs "Kente" vs "Kɛntɛ"
- **Tone matters**: Twi is tonal (ba = "come" vs "say")

### **Implementation Strategy**

#### **1. Hybrid Translation Layer**

```javascript
// services/translationService.js
class GhanaNLP {
  constructor() {
    // Primary: Google Translate (limited African language support)
    // Fallback: GhanaNLP.org API (open source Ghanaian NLP)
    // Tertiary: Rule-based for market terms
    this.ghanaNLPUrl = 'https://api.ghananlp.org/translation/v1';
  }

  async translateToEnglish(text, sourceLang) {
    // Try GhanaNLP first (better for local context)
    try {
      const response = await fetch(`${this.ghanaNLPUrl}/translate`, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'Authorization': `Bearer ${process.env.GHANA_NLP_KEY}`
        },
        body: JSON.stringify({
          text: text,
          source_language: sourceLang,
          target_language: 'en'
        })
      });

      const data = await response.json();
      return data.translation;
    } catch (e) {
      // Fallback to Google with custom glossary
      return await this.googleTranslateWithGlossary(text, sourceLang);
    }
  }

  // Specialized market terminology dictionary
  getMarketGlossary() {
    return {
      'tw': {
        'tɔ': 'buy',
        'tɔn': 'sell',
        'boɔ': 'price',
        'dɛ': 'good/nice',
        'fɛ': 'take',
        'ka': 'speak/say',
        'sika': 'money',
        'ton': 'reduce (price)',
        'ma me': 'give me'
      },
      'ga': {
        'shwɛ': 'buy',
        'shɔ': 'sell',
        'shishi': 'money',
        'lɛ': 'it is',
        'baŋŋ': 'good',
        'daŋŋ': 'bad'
      },
      'pidgin': {
        'chaw': 'expensive',
        'kpa': 'cheap',
        'dash': 'bonus/free',
        'how much': 'price',
        'i beg': 'please/negotiate',
        'sharp': 'quickly',
        'chop money': 'food money'
      }
    };
  }

  // Detect code-switching and segment
  async segmentMixedLanguage(text) {
    const prompt = `
      Analyze this Ghanaian text. Identify language segments and translate to English.
      Text: "${text}"

      Return JSON:
```

#### **3. SMS Character Optimization**

Since SMS is limited to 160 chars (or 70 for Unicode):

```javascript
function optimizeSMS(text, language) {
  // Use GSM-7 encoding where possible (160 chars)
  // Avoid Unicode for common languages if possible

  const replacements = {
    'tw': {
      'ɛ': 'e', // Simplify characters for SMS
      'ɔ': 'o',
      'ŋ': 'ng'
    },
    'ga': {
      'ɛ': 'e',
      'ɔ': 'o',
      'ŋ': 'ng'
    }
  };

  let optimized = text;
  if (replacements[language]) {
    Object.entries(replacements[language]).forEach(([char, repl]) => {
      optimized = optimized.replace(new RegExp(char, 'g'), repl);
    });
  }

  // Split long messages
  if (optimized.length > 160) {
    return splitSMS(optimized);
  }

  return [optimized];
}
```

## 21. DISPUTE RESOLUTION WORKFLOW (The "Abusuapanin" Model)

### **Cultural Context**
In Ghana, disputes are often resolved by **elders (Abusuapanin)** or **market queens**, not courts. Your platform
must mirror this.

### **The 3-Tier System**

```javascript
// models/DisputeResolution.js
class DisputeResolution {
  constructor() {
    this.tiers = {
      AUTOMATED: 'automated',    // AI + rules (80% of cases)
      COMMUNITY: 'community',    // Peer mediation (15%)
      ARBITRATION: 'arbitration' // Human expert (5%)
    };
  }

  async handleDispute(orderId, reason) {
    const order = await db.orders.findById(orderId);

    // Tier 1: Automated (Simple cases)
    if (this.isAutomatable(reason)) {
      return await this.automatedResolution(order, reason);
    }

    // Tier 2: Community Mediation
    if (order.amount < 500) { // GHS 500 threshold
      return await this.communityMediation(order);
    }

    // Tier 3: Expert Arbitration
    return await this.expertArbitration(order);
  }

  async automatedResolution(order, reason) {
    const rules = {
      'item_not_received': {
        check: async () => {
          // Check if vendor provided tracking/proof
          const proof = await db.proofs.findOne({ order_id: order.id });
          return !proof;
        },
        action: 'refund_buyer'
      },
      'item_not_as_described': {
        check: async () => {
          // AI image comparison
          const similarity = await compareImages(
            order.listing_images,
            order.customer_photos
          );
          return similarity < 0.6; // 60% match threshold
        },
        action: 'partial_refund'
      },
      'wrong_item': {
        action: 'return_and_refund'
      }
    };

    const rule = rules[reason];
    if (rule && await rule.check()) {
      await this.executeAction(rule.action, order);
      return { status: 'resolved', method: 'automated' };
    }

    // Escalate if automated fails
    return this.communityMediation(order);
  }

  async communityMediation(order) {
    // Select 3 neutral vendors from same market (but different category)
    const mediators = await db.vendors.find({
      market: order.vendor_market,
      category: { $ne: order.vendor_category },
      trust_score: { $gt: 80 },
      is_mediator: true
    }).limit(3);

    // Create mediation case
    const caseId = await db.cases.create({
      order_id: order.id,
      mediators: mediators.map(m => m.id),
      evidence: {
        buyer_statement: await collectStatement(order.buyer_id),
        vendor_statement: await collectStatement(order.vendor_id),
        chat_logs: await getWhatsAppLogs(order.id),
        photos: order.dispute_photos
      },
      status: 'pending_votes'
    });

    // Notify mediators via WhatsApp
    for (const mediator of mediators) {
      await whatsapp.send(mediator.phone,
        `Mediation request: ${order.product_name} (GHS ${order.amount}). ` +
        `Review: ${process.env.URL}/mediate/${caseId}. ` +
        `Earn GHS 5 for fair decision.`);
    }

    // 48-hour voting window
    setTimeout(async () => {
      await this.tallyVotes(caseId);
    }, 48 * 60 * 60 * 1000);

    return { caseId, status: 'community_review' };
  }

  async tallyVotes(caseId) {
    const votes = await db.votes.find({ case_id: caseId });

    // Majority rules
    const buyerWins = votes.filter(v => v.for === 'buyer').length > 1;

    if (buyerWins) {
      await this.refund(orderId, 0.9); // 90% refund (10% platform fee kept)
      await this.updateReputation(order.vendor_id, -5);
    } else {
      await this.releasePayment(orderId);
      await this.updateReputation(order.buyer_id, -2); // Penalize frivolous disputes
    }

    // Pay mediators
    for (const vote of votes) {
      await creditWallet(vote.mediator_id, 5); // GHS 5 for participating
    }
  }

  async expertArbitration(order) {
    // For high-value disputes (>GHS 1000)
    // Assign human reviewer (Ghanaian staff who understands context)

    const expert = await assignExpert(order.category); // Food vs Crafts experts

    await db.cases.update(order.id, {
      status: 'expert_review',
      assigned_expert: expert.id
    });

    // Video call mediation (if possible)
    await scheduleMediationCall(order.buyer_id, order.vendor_id, expert.id);
  }
}
```

```javascript
const UNIT_ECONOMICS = {
  // Revenue per transaction
  take_rate: 0.05,        // 5% commission

  net_revenue: 0.035,     // 3.5% net

  // Costs

  ltv_months: 6.67,       // Average 6.67 months lifespan

  // Vendor metrics
  gmv_per_vendor_month: 800, // GHS 800 monthly sales
  platform_fee: 30,       // GHS 30 for premium tier

  calculateLTV: function() {
    const commissionRevenue = this.gmv_per_vendor_month *
                             this.net_revenue *
                             this.ltv_months; // GHS 186.67

    const subscriptionRevenue = this.platform_fee * this.ltv_months; // GHS 200

    return commissionRevenue + subscriptionRevenue; // GHS 386.67
  },

  calculateCACPayback: function() {
    const monthly_profit = (this.gmv_per_vendor_month * this.net_revenue) +
                          (this.platform_fee * 0.3); // 30% conversion to premium
    return this.cac / monthly_profit; // ~1.8 months
  }
};
```
